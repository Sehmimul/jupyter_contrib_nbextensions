To start you will need a folder with your grammar, antlr.js, and custom listener files.
Make sure your antlr.js and listener.js files import and use the tools generated by your grammer 
    For example: 
        import parser from './<grammer-name>Parser.js';

Run the commands:
$ antlr4 -Dlanguage=JavaScript <grammer-name>.g4
$ npm init --yes
$ npm i antlr4

Then modify the package.json file by adding the line:
"type": "module",

Finally run it with:
$ node antlr.js

Check result here:
http://localhost:1337/
